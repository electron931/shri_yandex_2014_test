Show Popup

Использование псевдоэлементов :before, :target

Плюсы

  • Используется только CSS3 (без Javascript)
  • Окно адаптивное
  • Ширина и высота зависят только от внутреннего контента

Минусы

  • Не поддерживается в IE8
  • Метки в адресной строке (можно использовать javascript вместо :target)
